# S365.Search.Admin.UI

This project is for the Universal Search Administration (React Application) of features like:

- Feature Content
- Boosts
- Blocks

This can be extended in the future to allow more self service functions to the customer e.g.:

- Reporting.
- Updating search suggestions.
- Updating synonyms.
- Testing relevancy changes.
- Viewing crawler statistics.
- Etc.

## Swagger

Run the project locally here to view the Swagger or run the swagger from the S365.Search.Admin.UI Public URL: [S365 Deployments (S365Dev)](https://dev.azure.com/Search365/Azure%20Search365%20Platform/_wiki/wikis/Azure-Search365-Platform.wiki/304/Deployment-Endpoints-Domain-DNS-names?anchor=s365-deployments-(s365dev))
